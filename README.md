个人的 vim 配置。

# 插件管理工具下载
```
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

如果是有代理，可以使用类似于下面的命令，
```
curl -x http://example.proxy.com:8080 -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```
